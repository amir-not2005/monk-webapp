# monk-webapp

## Goal

Create a monk mode web application to be used in groups

## Technologies

_Frontend and Backend division with API_

1. Frontend
   - React js
   - Tailwind CSS
   - Axios
2. Backend
   - Node js
3. API
   - Express js
   - JWT Auth, Roles
4. Database
   - PostgreSQL
